# project1
"Расписание Лицей Иннополис"

Это проект для удобного и быстрого просмотра расписания.
Он полезен и для учеников, и для преподавателей.
Сама идея возникла из-за неудобства и потребности в быстром просмотре расписания.
Этот телеграм бот позволяет сделать это быстро и без лишних действий.
В самом боте ты выбираешь класс и день недели расписание которого тебе нужно.


Проект мы делали вдвоем: Насыбуллина Зарина и Зинатуллина Азалия.
Азалия сделала расписание звонков(что тоже не мало важно) и презентацию для показа нашего проекта классу и преподавателю
Зарина сделала кнопки телеграм бота и логику которые они выполняют.

Если этим телеграм ботом будут пользоваться ученики лицея Иннополис, то в будущем мы можем добавить какие-то новые полезные функции для продвижения проекта.
